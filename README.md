# Url-shortener

단축 URL 생성하기 (https://url.stlee.kr){:target="_blank"}

줄이고 싶은 URL을 입력 후 생성 해보세요.
생성된 QR코드를 이용하여 모바일에서 바로 접근 하실 수 있습니다.

## Tech stack

- **Backend**

  - Node.js
  - Koa
  - Mongodb

- **Frontend**

  - React
  - React Router v6
  - Redux
  - Redux-Saga
  - Styled Components
  - TypeScript
